# Quote Slicer Docs

Documentation site for **Quote Slicer**, an accessory app to [Project V] (a Chinese
character learning and dictionary app). Quote Slicer processes Chinese quotes and
texts, segmenting them and linking words and phrases to their translations.

Built with SvelteKit and mdsvex.

## Development

```bash
npm install
npm run dev
```

For notes on internal build mechanisms (ToC generation, wiki-style article links),
see [docs/architecture.md](docs/architecture.md).
